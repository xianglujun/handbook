# 占位符列表

在go语言中，`Printf`函数有超过10个这样的转义符，Go程序成为`verb`. 主要使用的有以下:

| verb             | 描述                                                         |
| ---------------- | ------------------------------------------------------------ |
| `%d`             | 十进制整数                                                   |
| `%x`, `%o`, `%b` | 十六进制，八进制，二进制                                     |
| `%f`, `%g`, `%e` | 浮点数类型. 如 `3.141593`, `3.141592653589793`, `3.141593e+00` |
| `%t`             | 布尔型：true或false                                          |
| `%c`             | 字符(Unicode码点)                                            |
| `%s`             | 字符串                                                       |
| `%q`             | 带引号字符串或字符                                           |
| `%v`             | 内置格式的任何值                                             |
| `%T`             | 任何值的类型                                                 |
| `%%`             | 百分号本身                                                   |
