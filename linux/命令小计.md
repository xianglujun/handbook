## 通过curl在服务器测试接口情况
```sh
curl -X POST \
  http://172.16.1.184:9081/v1/mc/listMarketCap \
  -H 'Content-Type: application/json' \
  -d '{}'
```

## 查看端口的监听状况
```sh
lsof -i:9081
```

## 查看CUP的占用情况
```sh
top

shift + p; shift + m; 可以按照不同的情况进行排序
```

## 查看某个端口的链接情况
```sh
netstat -apn | grep 9081
```

## centos 关闭命令行报警声音
```
vi /etc/inputrc

# 去掉注释
set bell-style none

# 修改值: none, visible,audible
set bell-style visible
```
