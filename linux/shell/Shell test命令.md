# Shell test命令

Shell中的test命令用于检查某个条件是否成立, 它可以进行数值, 字符和文件三个方面的测试.

## 数值测试

| 参数    | 说明      |
|:----- |:------- |
| `-eq` | 等于则为真   |
| `-ne` | 不等于则为真  |
| `-gt` | 大于则为真   |
| `-ge` | 大于等于则为真 |
| `-lt` | 小于则为真   |
| `-le` | 小于等于则为真 |

```shell
num1=100
num2=100
if test $[num1] -eq $[num2]
then
    echo '两个数相等！'
else
    echo '两个数不相等！'
fi
```
